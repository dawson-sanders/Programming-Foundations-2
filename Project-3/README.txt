PURPOSE:
- The program allows users to choose between a recursive binary search method or a iterative binary search methods to search for pinball machines. The recursive search method allows users to search for pinball machine titles. The iterative search method allows users to search by the year the pinball machine was made. 

TASKS:
- Create a Machine class that contains attributes about pinball machines 
- Read comma seperated data from an input file and store in machine vector
- Design the user interface
- Devolop methods for both recursive and iterative binary search

USER INSTRUCTIONS:
- Follow the command line interface prompts
